# MindScope AI

## Overview
**MindScope AI** is an NLP-based system designed to detect and classify mental health conditions from text inputs.  
Using the *Sentiment Analysis for Mental Health* dataset from Hugging Face, it explores how linguistic cues reflect emotional and psychological states.

The project compares multiple modeling paradigms—**TF-IDF + Naive Bayes**, **Word2Vec + Neural Network**, and **Fine-tuned BERT models**—to determine which approach most effectively captures nuanced mental health expressions.

Future work includes integrating the trained model into a **Flask web application** for real-time mental health text analysis.
